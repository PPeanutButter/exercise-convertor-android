# 导入Excel、json题库教程
> 支持任意版本的excel文件
> 仅支持人工导入获取的json文件

## 导入流程
- 下载插件(手机端)
- 准备您的excel题库
- 使用插件导入您的excel、json题库
- 额外的一些有用的小技巧

## 1. 安装POI插件
在release页面下载。

## 3. 导入处理好的excel题库
进入到POI插件中,在弹窗中选择Excel文件(`.xls` `.xlsx`)后等待软件处理完毕即可。

你可以使用任何excel小技巧，或者出几块钱找人工导入。
